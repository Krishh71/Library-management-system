Library Management System
A simple Python program to manage a library. You can add, display, borrow, and return books.

Setup:
Create a file named library_catalog.txt in the same folder as your code.
Add some book titles to it (one per line).

How to Run
Run this command in your terminal:
python vityarthi.py

Menu Options
D: Display all books

I: Issue (borrow) a book

A: Add a new book

R: Return a book

Q: Quit

Note: All new books are saved automatically to library_catalog.txt.
